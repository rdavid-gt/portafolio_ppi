Portafolio: Programación para Internet

Este portafolio contiene las prácticas, examen de medio término y una página acerca de mi realizados 
para la materia Programación para Internet del periodo Enero-Mayo 2026. Dichas prácticas son relacionadas 
con el uso de HTML, CSS, JS, entre otros.

1. About me: Una página centrada en mis conocimientos.
2. Examen MT: Página que contiene un temporizador entre dos fechas, una a elección y otra el día que se use.
3. Práctica 2: Página que hecha solo con HTML, ejemplos de su sintaxis como formularios, etiquetas,
               párrafos y títulos.
4. Práctica 3: Páginas que contienen una tabla, varias imágenes a diferentes tamaños, estilos y enlaces para
               regresar o ir a páginas externas.
5. Práctica 4: Páginas que contienen JavaScript para realizar varias operaciones como factorial (recursivamente),                    realizar equivalencias de calificaciones y llenar un formulario que utilice esa información para                      modificar el contenido de la misma página.
6. Práctica 5: Página que usa JavaScript junto con un formulario para crear una tabla con renglones que
               alternan estilos.
7. Práctica 6: Página que contiene contenido multimedia (video, audio, video de YouTube, componente de
               geolocalización, post de Twitter y de Facebook).
